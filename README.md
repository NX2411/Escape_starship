## 게임 실행 방법
- GameFile 폴더의 탈출!스타십우주비행선.zip 파일 다운로드
- 다운로드 한 파일 압축 해제
- 탈출!스타십 우주비행선.exe 파일 실행
- **해당 폴더에 있는 다른 파일 삭제 금지**
